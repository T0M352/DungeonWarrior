# DungeonWarrior
A game made in Java using the Libgdx library based on the system of procedural dungeons. The project utilized:
Box2D
Tiled
TrueType font
Scene2D
Ashley ECS
The game allows for endless gameplay thanks to the system of procedurally generated levels.
